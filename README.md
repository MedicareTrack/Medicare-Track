## Run Locally


Clone the project

```bash
  gh repo clone MedicareTrack/Medicare-Track
```


Go to the project directory

```bash
  cd MedicareTrack
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```



## Tech Stack

* [React](https://reactjs.org/)
