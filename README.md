## Run Locally

Clone the project

```bash
  gh repo clone MedicareTrack/Medicare-Track
```


Go to the project directory

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
